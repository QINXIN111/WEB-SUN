# 交互式太阳系引导页

这是一个基于 Three.js 构建的 3D 交互式太阳系可视化页面，展示了太阳系中主要天体的相对位置和运动轨迹。

## 功能特性

### 3D 可视化
- 真实比例展示太阳及八大行星
- 行星按照相对速度绕太阳公转
- 土星具有独特的行星环效果
- 星空背景增强空间感

### 交互功能
- **鼠标悬停**: 将鼠标悬停在任意行星上可查看其详细信息（名称、描述、半径、温度）
- **视角控制**: 拖拽鼠标可旋转观察角度
- **缩放功能**: 使用滚轮可放大缩小视图
- **重置视图**: 点击"重置视图"按钮可恢复默认视角
- **速度切换**: 点击"切换速度"可在正常速度和加速模式间切换

### 视觉效果
- 发光特效和后期处理（辉光效果）
- 扫描线和噪点叠加效果，营造科技感界面
- 响应式设计，适配不同屏幕尺寸
- 深色主题减少眼部疲劳

### 音频功能
- 集成音乐播放器
- 自动播放背景音乐
- 可调节音量大小

## 技术实现

- **核心框架**: Three.js v0.136.0
- **UI 框架**: Tailwind CSS
- **音频播放**: APlayer
- **后处理**: UnrealBloomPass 实现辉光效果
- **性能优化**: 
  - 限制设备像素比避免高分屏渲染压力
  - 减少轨道线和星空的几何细节
  - 使用静态噪点纹理替代动态滤镜

## 使用方法

直接在浏览器中打开 `太阳系引导页.html` 即可体验。

## 控制说明

- **鼠标左键拖拽**: 旋转视角
- **滚轮**: 缩放视图
- **右键拖拽**: 平移视角
- **重置视图按钮**: 恢复初始视角
- **切换速度按钮**: 在正常速度和5倍速之间切换
- **百度一下按钮**: 跳转到百度首页

## 注意事项

- 页面包含自动播放音频，调节音量请使用右下角的音乐播放器控件
- 在移动设备上可通过触摸操作控制视角
- 如遇到性能问题，可尝试缩小浏览器窗口或关闭其他应用程序
